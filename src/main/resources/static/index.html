Help Center

Quick Troubleshooting

Products are not loading

Check the API status at the top. If it shows Offline, confirm the backend is running on port 8080 and refresh.

Orders do not save

Make sure quantity and total price are valid numbers. Also verify customer and product selections are populated.

User creation fails

Ensure the role is set and the email is valid. If the response still fails, check server logs for validation errors.

Contact Support

Reach out with the error message shown in the alert banner for faster response.

System Checklist

  • Backend running on port 8080
  • Database connected and seeded
  • Browser cache cleared after updates
  • Role headers present for restricted actions
